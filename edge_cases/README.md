# edge_cases — твои собственные картинки для демо

Сюда клади **любые** изображения, которые хочешь прогнать через классификатор
(adversarial-примеры, пограничные случаи, реальные кейсы). Ноутбук
`notebooks/usecase_demo.ipynb` обработает **все** файлы из этой папки.

## Как разметить (опционально)

Чтобы картинка попала в сводную таблицу метрик, положи её в подпапку с меткой:

```
edge_cases/
  safe/      # ожидается «безопасно» (allow)
  unsafe/    # ожидается «небезопасно» (review/block)
  <в корне>  # без метки — будет показана и обработана, но не учтена в метриках
```

Поддерживаются: `.png .jpg .jpeg .webp .bmp .gif`.
